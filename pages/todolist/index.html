Fazer a atividade prática

1. Adicionar um novo link na nav da página principal, que irá direcionar para uma página de TODO.

2. Criar a nova página de TODO, utilizando grid, onde as rows da grid serão para o header, main e footer.

3. Na main da nova página, utilizar um form que terá um input de texto para o título da tarefa, um textarea para a descrição da tarefa, e um botão para enviar os dados do formulário.

4. Ao clicar no botão de enviar, adicionar esse texto à uma lista do localStorage.

5. Mostrar todos os itens do localStorage na página. Se recarregar a página, deve apresentar os itens já salvos.

6. Limpar os inputs após salvar.

Para estilizar o placeholder, utilizar o pseudo elemento ::placeholder

Para exibir as quebras de linha no parágrafo de descrição, utilizar o estilo white-space: pre-wrap;
